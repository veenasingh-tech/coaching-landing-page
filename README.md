# 🎓 Coaching Center Landing Page

A professional, high-converting landing page designed for coaching institutes or online tutors. This project focuses on clean UI and clear call-to-action buttons.

## 🚀 Live Demo
Check out the website here: https://scintillating-buttercream-9fd5f7.netlify.app/

## ✨ Key Features
* **Hero Section:** Eye-catching headline with a clear "Enroll Now" button.
* **Courses Offered:** Displaying different subjects or batches clearly.
* **Responsive Layout:** Optimized for mobile, tablet, and desktop screens.
* **Contact Form:** Simple lead generation section for students to inquire.
* **Testimonials:** Space for student feedback to build trust.

## 🛠️ Tech Stack
* **HTML5:** For semantic and structured web content.
* **CSS3:** For modern styling and responsiveness (using Flexbox/Grid).
* **JavaScript:** For smooth scrolling and interactive elements.
